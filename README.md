#  Automation Project For Wunder Mobility 

This project is related to an assignment by Wunder Mobility

## Getting Started

Project is based on Mobile and Backend automation so following software should be installed prior running tests.

## Prerequisites
 
- Java Development Kit (JDK)
- Android SDK
- Node.js
- Webdriver
- Appium
- Maven
- IntelliJ IDEA
- Android Device

## Instructions 
 Real android device required to perform tests, Please set desired capablitlies of your device in utility.BaseTest class. 


## Running the tests
    mvn clean install 
    RunTest.java
